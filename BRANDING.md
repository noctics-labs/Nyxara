# Nyxara Brand

Visual identity, color palette, and usage guidelines for Nyxara.

## Wordmark

Nyxara, set in a serif display face. Always with the capitalised "N".

When abbreviating internally, use `nyx` for filenames and `NYX` for env vars.

## Colors

| token | hex | use |
|---|---|---|
| Nebula Violet | `#7b4bd8` | primary accent |
| Deep Cosmos | `#1a0f3d` | background, dark surfaces |
| Star Cream | `#fff8e7` | text on dark, focal highlights |
| Solar Amber | `#ffd87a` | secondary highlights |
| Void | `#040214` | absolute background |
| Mist Violet | `#a87bff` | secondary text on dark |

## Logo files

- `assets/nyxara-logo.svg` — square 1:1 mark, use for avatars and tile thumbnails
- `assets/nyxara-banner.svg` — wide banner with wordmark, use on README and social cards

## Voice

Quiet, exploratory, no hype. "Discover", "scan", "drift", not "moon", "ape", "10x".

## Don't

- do not stretch the wordmark
- do not place the mark on busy photographic backgrounds
- do not recolour the mark outside this palette
